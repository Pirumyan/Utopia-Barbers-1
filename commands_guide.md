# Руководство по командам Telegram-бота (Barber Bot) / Բոտի հրամանների ուղեցույց / Bot Commands Guide

Здесь описаны все команды, которые доступны в боте. Обычные пользователи не имеют доступа к командам Администратора.
Այստեղ նկարագրված են բոտում հասանելի բոլոր հրամանները: Սովորական օգտատերերը մուտք չունեն ադմինիստրատորի հրամաններին:
Here are all the commands available in the bot. Regular users do not have access to Administrator commands.

---

## 👥 Команды для Клиентов / Հաճախորդների հրամաններ / Client Commands

Всё взаимодействие для клиентов построено на удобном меню с кнопками. Им не обязательно вводить команды вручную, но они существуют.
Հաճախորդների հետ փոխազդեցությունը կառուցված է հարմարավետ կոճակներով մենյուի վրա: Հրամաններ մուտքագրել պարտադիր չէ, բայց դրանք կան:
Client interaction is built on a convenient button menu. They don't have to enter commands manually, but they exist.

- `/start` 
  - 🇷🇺 Запустить бота и открыть главное меню.
  - 🇦🇲 Գործարկել բոտը և բացել գլխավոր մենյուն:
  - 🇬🇧 Start the bot and open the main menu.

---

## 👑 Команды Администратора / Ադմինիստրատորի հրամաններ / Admin Commands

Эти команды работают **только** у администраторов (указанных в `ADMIN_IDS`).
Այս հրամանները գործում են **միայն** ադմինիստրատորների համար (նշված են `ADMIN_IDS`-ում):
These commands work **only** for administrators (specified in `ADMIN_IDS`). 

### 1. Просмотр расписания / Ժամանակացույցի դիտում / View Schedule
- `/schedule` (или /shedule)
  - 🇷🇺 Показывает список всех записей клиентов на ближайшие 3 дня.
  - 🇦🇲 Ցույց է տալիս հաճախորդների բոլոր գրանցումները առաջիկա 3 օրվա համար:
  - 🇬🇧 Shows a list of all client appointments for the next 3 days.

### 2. Создание или изменение рабочего дня / Աշխատանքային օրվա ստեղծում կամ փոփոխում / Create or Modify Workday
Генерирует свободные слоты на выбранный период. / Ստեղծում է ազատ տեղեր ընտրված ժամանակահատվածի համար: / Generates free slots for the selected period.

- `/startday HH:MM HH:MM` (на сегодня / այսօրվա համար / for today)
  - 🇷🇺 **Пример:** `/startday 12:00 18:00` (Бот создаст слоты с 12.00 до 18.00 на сегодня).
  - 🇦🇲 **Օրինակ:** `/startday 12:00 18:00` (Բոտը կստեղծի տեղեր այսօր ժամը 12:00-ից մինչև 18:00):
  - 🇬🇧 **Example:** `/startday 12:00 18:00` (The bot will create slots from 12:00 to 18:00 for today).

- `/startday DD.MM HH:MM HH:MM` (на конкретный день / կոնկրետ օրվա համար / for a specific day)
  - 🇷🇺 **Пример:** `/startday 15.03 10:00 15:00`
  - 🇦🇲 **Օրինակ:** `/startday 15.03 10:00 15:00`
  - 🇬🇧 **Example:** `/startday 15.03 10:00 15:00`

### 3. Блокировка времени (занять слоты) / Ժամանակի արգելափակում / Block Time (Busy)
Делает слоты недоступными для записи. / Տեղերը դարձնում է անհասանելի գրանցման համար: / Makes slots unavailable for booking.

- `/busy HH:MM` или `/busy DD.MM HH:MM` (1 слот / 1 տեղ / 1 slot)
  - 🇷🇺 **Пример:** `/busy 14:30` (на сегодня) или `/busy 15.03 19:30`
  - 🇦🇲 **Օրինակ:** `/busy 14:30` (այսօրվա համար) կամ `/busy 15.03 19:30`
  - 🇬🇧 **Example:** `/busy 14:30` (for today) or `/busy 15.03 19:30`

- `/busy HH:MM HH:MM` или `/busy DD.MM HH:MM HH:MM` (период / ժամանակահատված / period)
  - 🇷🇺 **Пример:** `/busy 19:30 21:00` (на сегодня) или `/busy 15.03 19:30 21:00`
  - 🇦🇲 **Օրինակ:** `/busy 19:30 21:00` (այսօրվա համար) կամ `/busy 15.03 19:30 21:00`
  - 🇬🇧 **Example:** `/busy 19:30 21:00` (for today) or `/busy 15.03 19:30 21:00`

### 4. Разблокировка времени (освободить слоты) / Ժամանակի ապաշրջափակում (ազատել տեղերը) / Unblock Time (Unbusy)
Делает слоты снова доступными. / Տեղերը կրկին հասանելի է դարձնում: / Makes slots available again.

- `/unbusy HH:MM` или `/unbusy DD.MM HH:MM` (1 слот / 1 տեղ / 1 slot)
  - 🇷🇺 **Пример:** `/unbusy 14:30` (на сегодня) или `/unbusy 15.03 19:30`
  - 🇦🇲 **Օրինակ:** `/unbusy 14:30` (այսօրվա համար) կամ `/unbusy 15.03 19:30`
  - 🇬🇧 **Example:** `/unbusy 14:30` (for today) or `/unbusy 15.03 19:30`

- `/unbusy HH:MM HH:MM` или `/unbusy DD.MM HH:MM HH:MM` (период / ժամանակահատված / period)
  - 🇷🇺 **Пример:** `/unbusy 19:30 21:00` (на сегодня) или `/unbusy 15.03 19:30 21:00`
  - 🇦🇲 **Օրինակ:** `/unbusy 19:30 21:00` (այսօրվա համար) կամ `/unbusy 15.03 19:30 21:00`
  - 🇬🇧 **Example:** `/unbusy 19:30 21:00` (for today) or `/unbusy 15.03 19:30 21:00`

### 5. Отмена клиентской записи / Հաճախորդի գրանցման չեղարկում / Cancel Client Appointment
Отменяет бронь клиента и присылает ему уведомление об отмене. / Չեղարկում է հաճախորդի գրանցումը և ծանուցում ուղարկում: / Cancels the client's booking and sends them a cancellation notice.

- `/cancel HH:MM` (на сегодня / այսօրվա համար / for today)
  - 🇷🇺 **Пример:** `/cancel 16:00`
  - 🇦🇲 **Օրինակ:** `/cancel 16:00`
  - 🇬🇧 **Example:** `/cancel 16:00`

- `/cancel DD.MM HH:MM` (на конкретный день / կոնկրետ օրվա համար / for a specific day)
  - 🇷🇺 **Пример:** `/cancel 15.03 10:00`
  - 🇦🇲 **Օրինակ:** `/cancel 15.03 10:00`
  - 🇬🇧 **Example:** `/cancel 15.03 10:00`

### 6. Назначение выходного / Հանգստյան օրվա նշանակում / Set Day Off
Удаляет все слоты и клиентские записи (оповещая клиентов) на весь день. / Հեռացնում է բոլոր տեղերն ու հաճախորդների գրանցումները (ծանուցելով նրանց) ողջ օրվա համար: / Deletes all slots and client appointments (notifying clients) for the entire day.

- `/dayoff DD.MM`
  - 🇷🇺 **Пример:** `/dayoff 14.03`
  - 🇦🇲 **Օրինակ:** `/dayoff 14.03`
  - 🇬🇧 **Example:** `/dayoff 14.03`

### 7. Создание стандартного рабочего дня / Ստանդարտ աշխատանքային օրվա ստեղծում / Create Standard Workday
Отменяет выходной и генерирует стандартные пустые слоты (с 10:00 до 21:00). / Չեղարկում է հանգստյան օրը և ստեղծում ստանդարտ ազատ տեղեր (10:00-ից մինչև 21:00): / Cancels a day off and generates standard free slots (from 10:00 to 21:00).

- `/workday DD.MM`
  - 🇷🇺 **Пример:** `/workday 14.03` 
  - 🇦🇲 **Օրինակ:** `/workday 14.03`
  - 🇬🇧 **Example:** `/workday 14.03`
